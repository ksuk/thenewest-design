Browser extensions

6 best browser extensions for developers, designers and makers

ColorZilla

ColorZilla includes a Color Picker, Eye Dropper, Gradient Generator and many additional advanced color tools.

Fontface Ninja

Fontface Ninja is a free browser extension that let you inspect, try, buy and bookmark any fonts on any website

CSS Peeper

Smart CSS viewer tailored for Designers

JSON Viewer

The most beautiful and customizable JSON/JSONP highlighter that your eyes have ever seen

Wappalyzer

Wappalyzer is a cross-platform utility that uncovers the technologies used on websites. It detects content management systems, ecommerce platforms, web frameworks, server software, analytics tools and many more.

Accessibility Insight

Solve accessibility issues before they reach your customers
