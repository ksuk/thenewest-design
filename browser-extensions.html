Browser extensions

開発者やデザイナーが使用するのに最適なブラウザ拡張機能

ColorZilla

ColorZillaには、カラーピッカー、アイドロッパー、グラデーションジェネレーター、およびその他の多くの高度なカラーツールが含まれています。

Fontface Ninja

Fontface Ninjaは、任意のウェブサイトのフォントを検査、試用、購入、ブックマークできる無料のブラウザ拡張機能です。

CSS Peeper

デザイナー向けにカスタマイズされたスマートCSSビューアーです。

JSON Viewer

　今まで最も美しく、カスタマイズ可能なJSON/JSONPのハイライトツールです。

Wappalyzer

Wappalyzerは、ウェブサイトで使用されているテクノロジーを明らかにするクロスプラットフォームユーティリティです。コンテンツ管理システム（CMSを）、Eコマースプラットフォーム、フレームワーク、サーバーソフトウェア、分析ツールなどを検出します。

Accessibility Insight

ユーザーが使用する前にアクセシビリティの問題を解決しましょう。
