Generators

CSS、カラー、グラデーション、グリッドを発見するために最適なジェネレーターツール

Blobmaker

わずか数秒でSVG形状を作成できます。

Colormind

Colormindはディープラーニングを用いたカラースキームジェネレータです。写真、映画、人気の芸術からカラースタイルを学ぶことができます。

Replace Cover

Spotifyはプレイリストのためのクールなアルバムアートを作成できます。

CSS Gradient

CSS Gradientを使用すると、ウェブサイトのグラデーション背景を作成することができます。

Duotones

写真のためのDuotoneペアリングとジェネレーターです。

cssgr.id

Cssgr.idはCSSグリッドレイアウトのための定型的なコードを生成するための対話型ツールです。

Random user generator

ランダムなユーザーデータを生成するための無料のオープンソースAPIです。人用のLorem Ipsumのようなものです。

Twisted Polygons

プロットするためのポリゴンSVGを生成するためのプロジェクトです。

CSS Grid Generator

css-gridを使用して、独自のグリッドを生成するジェネレーターです。

Avatar generator

現代のための遊び心あふれるアバタージェネレータです。

Stubborn

ジェネレーターを使用し、キャラクターを無料でデザインする機会を得ましょう。

TwitPile

Twitterユーザーとリストからタイル状の画像を作成するデザインツールです。Twitterでサインアップしましょう。

GetWaves

次世代のウェブデザインのために、ユニークなSVG Waveを作成する無料のSVGウェーブジェネレーターです。曲線を選択し、複雑さを調整し、ランダム化します。
